# 音声授乳タイマー

声だけで操作できる授乳タイマーアプリです。赤ちゃんを抱っこしながらでも、手を使わずにタイマーを操作できます。

## 概要

「右」「スタート」などの音声コマンドでタイマーを開始・停止・記録できます。授乳記録は端末内に最大50件保存され、日時・左右・時間を一覧で確認できます。

## 機能

- 音声コマンドでタイマーを操作（手が離せない授乳中に便利）
- 左右どちらの授乳かを記録
- 授乳ログを最大50件保存
- ログの個別削除・全削除

## 音声コマンド一覧

| コマンド | 動作 |
|---------|------|
| 「右」 | 右おっぱいを選択 |
| 「左」 | 左おっぱいを選択 |
| 「スタート」 | タイマー開始 |
| 「ストップ」 | タイマー一時停止 |
| 「完了」 | 記録を保存 |
| 「リセット」 | タイマーをリセット |

## 使い方

1. アプリを起動する
2. 「最初に押してください」ボタンをタップして音声入力を有効にする
3. 「右」または「左」と話しかけて授乳する側を選ぶ
4. 「スタート」と話しかけてタイマーを開始する
5. 授乳が終わったら「完了」と話しかけて記録を保存する

## 必要な環境

- iOS 16.0 以上
- Xcode 15 以上（ビルドする場合）
- マイクの使用許可
- 音声認識の使用許可（日本語）

## 技術仕様

| 項目 | 内容 |
|------|------|
| 言語 | Swift / SwiftUI |
| 音声認識 | Speech フレームワーク（`SFSpeechRecognizer`） |
| マイク入力 | AVFoundation（`AVAudioEngine`） |
| データ保存 | `UserDefaults`（端末内のみ・外部送信なし） |
| 外部通信 | なし |

## プライバシー

- マイクと音声認識の権限のみ使用します
- 収集したデータは端末内にのみ保存され、外部には一切送信されません
- 授乳記録（時刻・左右・時間）は `UserDefaults` に保存されます
